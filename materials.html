Materials

  1. Choose material that you want to rewrite
  2. Rewrite data
  3. Save new file as one of sheets.js/fillers.js/strips.js
  4. Delete original file from location C/.../calculator/javascript_materials
  5. After file is downloaded put it into folder C/.../calculator/javascript_materials
  6. In case of sheets sizes are: 0.5, 0.6, 0.8, 1, 2, 3, 4, 5
  7. In case of strips sizes are: 2.5, 3.2, 4.8, 6.4, 7.2
  8. !!! Some options are scrollable !!!
